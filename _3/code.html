艾灸疗法 (Moxibustion Therapy)

温经散寒 扶阳固脱 消瘀散结

摘要：艾灸疗法是以艾绒为主要材料，点燃后放置在腧穴或病变部位，通过温热刺激及药物作用，达到防治疾病目的的一种中医外治法。

check_circle

适应证

  • • 虚寒性痛证（痛经、胃痛）
  • • 风寒湿痹（关节酸痛）
  • • 脏腑虚寒证（泄泻、遗尿）
  • • 气虚下陷（脱肛、子宫脱垂）
warning

禁忌证

  • • 实热证、阴虚发热者
  • • 面部、大血管处及黏膜附近
  • • 孕妇腰骶部及下腹部
  • • 极度疲劳、大醉、大怒、大恐
precision_manufacturing

操作要点

01

环境准备

保持室内空气流通，调节室温适宜，遮挡屏风保护隐私。

02

穴位定位

根据医嘱选取穴位，指导患者取舒适体位并充分暴露施灸部位。

03

施灸操作

点燃艾条，距皮肤3-5cm。灸至局部皮肤红晕、温热舒适为度。

visibility

护理观察

观察施灸部位皮肤颜色，询问患者温热感。注意神志变化，防止“晕灸”。

history_edu

健康教育

  • • 嘱患者灸后半小时内禁冷水沐浴。
  • • 饮食宜清淡，忌辛辣生冷。
  • • 若出现小水泡可自行吸收。
tips_and_updates

注意事项

严格执行查对制度。施灸过程中严禁离开患者，及时清理艾灰，严防火灾。对于知觉迟钝者（如糖尿病患者），应以手护其局部皮温，防深度烫伤。
